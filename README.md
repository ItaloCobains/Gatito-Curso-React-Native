## Projeto

Simple aplicativo para android e ios escriot em react-native, simulando um pet shop.

## Requisitos

## Como rodar

Baixe o repositorio
```bash
    git clone https://github.com/ItaloCobains/Gatito-Curso-React-Native
```

Baixe os pacotes
```bash
    npm i
```
Ou
```bash
    yarn add
```

A rode o projeto com
```bash
    npm start
```
Ou
```bash
    yarn start
```
Ou
```bash
    expo start
```
